# Frontend - Retail Sales Management System

## Setup Instructions

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:3000`

3. Build for production:
```bash
npm run build
```

## Features

- Real-time search across customer name and phone number
- Multi-select filters for regions, gender, categories, tags, and payment methods
- Age range and date range filtering
- Sorting by date, quantity, and customer name
- Pagination with 10 items per page
- Responsive design

## Tech Stack

- React 18
- Vite
- Axios
- CSS3
